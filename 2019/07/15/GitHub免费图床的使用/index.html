GitHub免费图床的使用

教程
  • github
  • hexo

很多小伙伴都在用markdown写东西，却苦于博客图片无法正常显示，由于大多数博客使用的图片较少如果选择付费图床并不合算，所以自己搭建一个免费的图床成为最佳选择，本文主要介绍GitHub Page图床的搭建过程。

GitHub图床的搭建

1.登陆你的GitHub账号，新建一个 repositories

2.输入存储库的名字

3.选择wiki，create the first page

4.复制右下角链接

5.进入站点根目录，GitHub Bash,克隆刚刚复制的链接

6.将需要添加的图片复制到克隆好的文件(myblogimg.wiki)

7.cd到myblogimg.wiki

8.上传到图床

9.访问你上传的图片

https://raw.githubusercontent.com/wiki/UserName/PIC/图片1.png

UserName：你的github用户名

PIC：你新建的储存库

图片1.png：你要访问的图片

markdown中插入图片时输入以上链接即可
